# M01 — Fundamentos: O que é 45D Sprint?

> O sistema operacional para transformar lojas Shopify em máquinas de tráfego orgânico em 45 dias.

## Objectivo

Ao final deste módulo, o aluno compreende:
- O que é 45D Sprint e por que 45 dias
- As 4 fases e o que acontece em cada uma
- Como o sistema integra IA, automação e operações reais
- Como replicar este framework para novos clientes
